# Advisor Transition Wizard

A Next.js 14 application with Tailwind CSS designed to streamline the advisor transition process.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- ShadcnUI Components

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/username/advisor-transition-wizard.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

```
advisor-transition-wizard/
├── src/              # Source files

│   ├── components/  # Reusable components
│   └── lib/         # Utility functions

```
